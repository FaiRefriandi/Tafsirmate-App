# Tafsirmate App

Tafsirmate is a mobile application designed to help users learn basic Qur'an interpretation (tafsir) using an AI-powered assistant. The app provides a simple interface to read surahs, view ayat details, and interact with an AI chatbot for explanations.

## Overview

This project is built using native Android development.

Key features include:
- Surah and ayat listing
- Detailed ayat view
- AI chatbot for tafsir explanation
- Voice input and text-to-speech output
- Google Sign-In for user authentication
- Modern UI with fragments and navigation components

---

## Features

- Display list of surahs from the Qur'an
- View ayat details in a dedicated DetailFragment
- AI assistant for asking questions about tafsir
- Support for text and voice input
- Continuous conversation context
- Dynamic suggestion chips in chat UI
- Google Sign-In integration
- User profile page
- Splash screen and bottom navigation
  
---

## Tech Stack

- Android (Kotlin)
- XML layout
- ViewModel / LiveData
- Navigation Component
- OpenAI API
- TTS (TextToSpeech)
- SpeechRecognizer
- Firebase Authentication (Google Sign-In)
- OkHttp networking

---

## Installation

1. Clone the repository
```bash
git clone https://github.com/FaiRefriandi/Tafsirmate-App.git
```

2. Open the project in Android Studio.
3. Sync Gradle.
4. Add your API keys (OpenAI, Firebase config).
5. Build and run on an emulator or Android device.

---

## Requirements

- Android Studio Hedgehog or newer
- Minimum SDK version as defined in `build.gradle`
- OpenAI API key
- Google Firebase project (for Google Sign-In)
